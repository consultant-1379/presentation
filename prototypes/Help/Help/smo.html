Software Management Organizer
Software Management Organizer (SMO) manages all operations involved in software upgrades and licence key installations and facilitates the distribution of new software from the Ericsson release server to network elements.
CPI STORE

  • Visit the CPI Store for information regarding this application

  • Software Management Organizer, User Guide

  • For additional information see the System Admin Guide and Function Description


Main Functions

  • Backup administration, including transfer of backups of OSS for supported network elements.

  • Remote Software Upgrade

  • Software Distribution from OSS to network elements

  • Software Inventory, including comparison between network elements.

  • Hardware Inventory for Supported Network Elements
