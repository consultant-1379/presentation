View and Export SW Inventory
To export a Software Inventory for a Managed Element, complete the following procedure:
  1. Click the Inventory tab to get into Inventory view:

  2. Select one or more MEs.

    If ME not visible in Topology, it could be because :

    • ME version is not ECIM compliant.
    • ME is not added to topology.
    • User has no authority to view ME.

    For more information on how to add an ME to topology, refer to OSS Explorer (OEX) GUI User Guide Check the authority settings for the user. For more information about Authority Handling, refer to SHM, Software Hardware Manager System Administrator Guide

  3. Click Export.

  4. In the Export Inventory dialog specify the file name.
    5. The default file name is : InventoryExport_ yyyy-MM-dd'T'HHmm

    Note: The time stamp reflects server time not local time

  5. Confirm that the SW Inventory checkbox is selected.

  6. Click Export.

  7. Open or save the file.

    8. The export process runs in the background.
